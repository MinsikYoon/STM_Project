# 고추건조기 프로젝트


### 프로젝트 목적
- STM32보드 사용법 및 HAL드라이버 숙달

### 프로젝트 기간 : 2024. 04. ~ 2024. 05.

### 프로젝트 준비
- 사용 보드 : STM32F103C8T6
- 사용 언어 : C
- 사용 툴 : STM32CubeIDE
- 사용 하드웨어
  - 온도센서
  - FND 모듈
  - 스위치
  - OLED
  - LED
  - 버튼
  - 릴레이 모듈
